Terminal Maze Game

A minimalist, text-based maze game in Java. Navigate the maze, avoid enemies, and reach the exit — all in your terminal.

Features

Procedurally generated maze every time you reach the exit

Player represented by @

Exit represented by E

Enemy (X) spawns 1 second after your first move and follows you

Real-time WASD controls

Cross-platform support: works on Linux, macOS, and Windows terminals

Fully playable in terminal — no GUI required

Gameplay

Use W, A, S, D to move your player.

Avoid the enemy, which spawns after 1 second and follows you.

Reach the exit (E) to generate a new maze.

If an enemy touches you, the game restarts.

Installation

Clone the repository:
git clone https://github.com/yourusername/TerminalMazeGame.git

Navigate to the project folder:
cd TerminalMazeGame

Compile the Java code:
javac TerminalMazeGame.java

Run the game:
java TerminalMazeGame
